Rajah
======

Rajah is key-value store written in Java. Clinet-server interface depends on Redis-Protocol.
- SET(implemented)
- GET(implemented)
- MSET(implemented)
- MGET(implemented)
- DEL(implemented)
- EXISTS(implemented)
- KEYS
- RENAME
- RENAMENX

- EXPIRE
- EXPIREAT
- PERSIST
- PEXPIRE
- PEXPIREAT
- PTTL
- TTL
